ขั้นตอนที่ 2: สร้างโปรเจกต์ Blazor Web App ใน Visual Studio
ตอนนี้เราจะสร้างโปรเจกต์กันค่ะ มี 2 วิธีให้เลือก คุณชอบแบบไหนคะ?
วิธีที่ 1: สร้างผ่าน Visual Studio (แนะนำสำหรับมือใหม่)

เปิด Visual Studio 2022
สร้างโปรเจกต์ใหม่:

คลิกที่ "Create a new project"
ในช่องค้นหา พิมพ์ "Blazor"
เลือก "Blazor Web App" (ดูให้ดีว่าเป็น Web App นะคะ ไม่ใช่ Server หรือ WebAssembly)

Project name: พิมพ์ BlazorSample
Location: เลือกตำแหน่งที่จะบันทึกโปรเจกต์ (เช่น C:\Projects)
คลิก Next


เลือก Framework:

Framework: เลือก .NET 8.0 (Long Term Support)
Authentication type: None
เลือกตัวเลือกตามนี้:

✅ Interactive render mode: Auto (Server and WebAssembly)
✅ Interactivity location: Global
✅ Include sample pages (ติ๊กไว้)


คลิก Create


รอสักครู่ Visual Studio จะสร้างโปรเจกต์ให้


วิธีที่ 2: สร้างผ่าน Command Line (เหมือนในเอกสาร)
ถ้าคุณอยากลองแบบที่เอกสารสอนก็ทำได้ค่ะ:

เปิด Command Prompt หรือ PowerShell
ไปที่โฟลเดอร์ที่ต้องการสร้างโปรเจกต์ เช่น:

   cd C:\Projects

สร้างโปรเจกต์ด้วยคำสั่ง:

   dotnet new blazor -o BlazorSample

เปิดโปรเจกต์ใน Visual Studio:

   cd BlazorSample
   start BlazorSample.sln

วิธีที่ 2: Trust Certificate (แก้ไขถาวร - แนะนำ)
ถ้าคุณไม่อยากเห็นคำเตือนนี้อีก:

ปิดเบราว์เซอร์ก่อน

พิมพ์คำสั่งนี้:

   dotnet dev-certs https --trust

จะมีกล่องข้อความถามว่าต้องการติดตั้ง Certificate หรือไม่ → คลิก "Yes"
กลับไปที่ Visual Studio และรันโปรเจกต์อีกครั้ง (กด F5)

วิธีสร้าง Git Repository:
วิธีที่ 1: ใน Visual Studio (ง่ายสุด)

เปิด Visual Studio (โปรเจกต์ BlazorSample)
ดูที่เมนูบน เลือก Git → Create Git Repository
เลือก:

✅ Local repository only (ถ้าจะใช้แค่ในเครื่อง)
หรือ ✅ Push to GitHub (ถ้าต้องการอัปโหลดไป GitHub เลย)


คลิก Create and Push

Visual Studio จะสร้างไฟล์ .gitignore ที่เหมาะสมกับโปรเจกต์ .NET ให้อัตโนมัติค่ะ!

วิธีที่ 2: ใช้ Command Line

เปิด Command Prompt หรือ PowerShell
ไปที่โฟลเดอร์โปรเจกต์:

bash   cd C:\Projects\BlazorSample

สร้าง Git repository:

bash   git init

สร้างไฟล์ .gitignore สำหรับ .NET:

bash   dotnet new gitignore

Commit ครั้งแรก:

bash   git add .
   git commit -m "Initial commit - Blazor Sample project"

เชื่อมกับ GitHub (ถ้าต้องการ):

สร้าง Repository ใหม่บน GitHub ก่อน
แล้วรันคำสั่ง:



bash   git remote add origin https://github.com/YOUR_USERNAME/BlazorSample.git
   git branch -M main
   git push -u origin main

ไฟล์ที่ควร Ignore (.gitignore)
ไฟล์ .gitignore ที่ Visual Studio หรือ dotnet new gitignore สร้างให้จะมีสิ่งเหล่านี้อยู่แล้ว:

bin/ - ไฟล์ที่ compile แล้ว
obj/ - ไฟล์ชั่วคระระหว่าง build
.vs/ - การตั้งค่า Visual Studio
*.user - การตั้งค่าส่วนตัว
